Dasar-Dasar Pemrograman Java

CAPAIAN PEMBELAJARAN

[1] Setelah pembelajaran, mahasiswa dapat memahami pendahuluan java dan dapat membedakan java dengan bahasa pemrograman lainnya.
[2] Setelah pembelajaran, mahasiswa dapat membuat dan menerapkan variabel dan tipe data.
[3] Setelah pembelajaran, mahasiswa dapat menerapkan operator dalam program.
[4] Setelah pembelajaran, mahasiswa memahami konsep percabangan dan dapat membuat program dalam bahasa Java.
[5] Setelah pembelajaran, mahasiswa memahami konsep perulangan dan dapat membuat program dalam bahasa Java.
[6] Setelah pembelajaran, mahasiswa dapat memahami konsep fungsi dan dapat mengaplikasikan penggunaan fungsi pada bahasa Java.
