Dasar-Dasar Pemrograman Java

Pengenalan Bahasa Pemrograman Java

  • Mahasiswa mampu menerangkan pengertian Java dengan baik
  • Mahasiswa menerangkan sintaks Java dengan baik
  • Mahasiswa mampu mengeksekusi program Java dengan baik
B. Pengenalan Sintaks Java

Sebelum membahas lebih lanjut mengenai dasar-dasar pemrograman pada Java, pada bagian ini akan dijelaskan mengenai sintaks, yaitu berbagai bagian-bagian kode serta aturan dalam penulisan kode Java.

  • Penulisan Nama Class

    • Di baris paling awal terdapat perintah class NamaClass. Kode ini dipakai untuk menulis nama class ke dalam kode program kita. Nama class ini boleh ditulis bebas, tapi kebiasaan programmer java adalah menulisnya dalam format PascalCase, dimana setiap kata diawali dengan huruf besar (termasuk kata pertama) dan tanpa spasi.

    Penulisan nama class sebaiknya juga harus sama dengan nama file. Sebagai contoh, dalam kode program menggunakan nama class BelajarJava, maka nama file yang dipakai adalah BelajarJava.java. Jika menggunakan nama class lain, maka nama filenya juga ditulis sesuai NamaFile.java.

Penulisan Class dalam Program Java
Main.java
                                  
class Main {
    ...
    ...
}

Seluruh kode program untuk sebuah class berada di dalam tanda kurung kurawal adalah ‘kepunyaan’ dari class Main.

  • public static void main(String args[])

    • Kode program ditulis dalam bahasa Java memiliki struktur dasar tersendiri, misalnya diawali dengan public class namaFile serta memiliki perintah public static void main(String args[]). Berikut adalah definisi dari public static void main(String args[]).

    1. public: Berfungsi sebagai access modifier, yakni batasan akses dari sebuah kode program. Jika ditulis public, maka kode tersebut bisa diakses dari luar class. Nantinya ada beberapa access modifier lain seperti private dan protected.
    2. static: Menandakan bahwa ini adalah sebuah method yang bisa diakses langsung dari dalam class (tanpa harus membuat object).
    3. void: Artinya method ini tidak mengembalikan nilai.
    4. main: Ini merupakan nama dari method. Selain itu main adalah nama method khusus yang akan dibaca oleh Java compiler untuk memulai proses compile. Setiap aplikasi Java harus memiliki sebuah main method.
    5. (String args[]): Merupakan argument dari mai method, yang dipakai untuk ‘menangkap’ sebuah nilai ketika kode program dijalankan dari cmd.
Penulisan public static void main(String args[]) dalam Program Java
Main.java
                                  
class Main {
    public static void main(String args[])
        ...
        ...
}

Semua kode yang ada di antara kurung kurawal ini adalah kode milik public static void main(String args[]).

  • System.out.println()

    • Perintah System.out.println() adalah sebuah kode program untuk proses output, yakni menampilkan teks di layar. Nantinya ada beberapa perintah lain yang bisa dipakai untuk proses output seperti ini, dan juga untuk proses input (menerima nilai).

Cobalah contoh di bawah ini menggunakan compiler
Main.java
                                
class Main {
    public static void main(String args[]){
        System.out.println("Semangat!");
        System.out.println("Belajar Java Online");
        System.out.println("di Javatikom");
    }
}
Dasar-Dasar Pemrograman Java

Pada materi sebelumnya, kita sudah mempelajari penulisan class main dan public static void main(String args[]), dan kita menggunakan kode berikut untuk mencetak "Semangat", "Belajar Java Online", "di Javatikom" ke layar.

Salin kode pada editor ke compiler Trinket, kemudian jalankan!

                            
class Main {
    public static void main(String args[]){
        System.out.println("Semangat!");
        System.out.println("Belajar Java Online");
        System.out.println("di Javatikom");
    }
}
Komentar

Komentar adalah bagian dari kode program yang tidak akan dieksekusi saat program dijalankan, tetapi berguna untuk memberikan penjelasan atau dokumentasi pada kode tersebut. Dalam bahasa pemrograman Java, terdapat dua jenis komentar, yaitu:

  • Single-line comment
    • Single-line comment : Komentar ini dimulai dengan dua garis miring (//) dan berlaku untuk satu baris kode. Semua teks setelah // dianggap sebagai komentar dan tidak akan dieksekusi oleh program.
  • Multi-line comment
    • Multi-line comment : Komentar ini dimulai dengan /* dan diakhiri dengan //. Semua teks di antara / dan */ dianggap sebagai komentar dan tidak akan dieksekusi oleh program. Komentar ini dapat mencakup beberapa baris kode.

Cobalah contoh di bawah ini menggunakan compiler
Main.java
                                
class Main {
    public static void main(String args[]){
     /*
        komentar
        gaya javascript
        */
     //komentar gaya C++
    }
}
