H. Tipe Data Array

Tipe data array adalah tipe data yang terdiri dari kumpulan tipe data lain. Dalam bahasa Indonesia, array dikenal juga dengan istilah Larik. Dengan array, proses penyimpanan data ke dalam variabel menjadi lebih efisien dan mudah, terutama jika memiliki data dalam jumlah banyak.

Anggota atau isi dari array itu sendiri harus satu jenis tipe data, misalkan terdiri dari kumpulan angka bulat saja (integer), kumpulan karakter saja (char), maupun kumpulan angka pecahan saja (double). Di dalam bahasa Java, kita tidak bisa membuat 1 array dengan berbagai tipe data (harus 1 jenis saja).

Sebagai contoh, misalnya kita ingin menyimpan 5 buah nilai siswa. Tanpa array, maka harus menyiapkan 5 buah variabel:

                                      
int nilai1, nilai2, nilai3, nilai4, nilai5;

Jika menggunakan array, pendefinisian variabel cukup seperti ini:

                                      
int[] nilai = new int[5];

Atau bagaiman jika ingin 1000 nilai? tidak masalah:

                                      
int[] nilai = new int[1000];

Tentu jauh lebih efisien dibandingkan membuat 1000 buah variabel.

Cara Pembuatan Tipe Data Array

Ciri khas pembuatan tipe data array adalah dengan tanda kurung siku ” [ ] “. Tanda kurung siku ini ditempatkan setelah penulisan tipe data. Misalnya saya ingin membuat array nama yang terdiri dari tipe data String, maka perintahnya adalah:

                                      
string[] nama;

Atau jika ingin membuat array nilai dari tipe data char, maka penulisannya adalah:

                                      
char[] nilai;

Namun ini baru sebatas deklarasi saja, agar bisa digunakan kita harus sambung dengan proses instansiasi dan inisialisasi.

Deklarasi adalah istilah untuk menyebut proses pembuatan variabel, dimana kita meminta compiler Java agar menyiapkan tempat di memory komputer untuk sebuah variabel. Inisialisasi sendiri adalah proses pemberian nilai awal ke variabel tersebut.

Perintah deklarasi dan inisialisasi ini sudah sering kita pakai sebelumnya dan biasa disambung dalam 1 baris perintah, seperti:

                                        
  String var1 = "Belajar Java Online"

Namun khusus untuk array, perlu tambahan instansiasi, yakni proses pembuatan object, secara sederhana perintah instansiasi ditandai dengan keyword new.

Cobalah kode program ini menggunakan compiler
Main.java
                                            
class Main {
  public static void main(String args[]) {
    // Deklarasi Tipe Data dan Variabel

    int[] nilai;

    nilai = new int[5];

    nilai[0] = 23;
    nilai[1] = 50;
    nilai[2] = 34;
    nilai[3] = 78;
    nilai[4] = 90;

    System.out.println(nilai[0]);
    System.out.println(nilai[1]);
    System.out.println(nilai[2]);
    System.out.println(nilai[3]);
    System.out.println(nilai[4]);
  }
}
Salinlah kode di bawah ini, kemudian jalankan!

Cara Singkat Pembuatan Array

Bahasa Java juga menyediakan cara alternatif untuk pembuatan array, salah satunya dengan menggabungkan proses deklarasi dan instansiasi.

Cobalah kode program ini menggunakan compiler
Main.java
                                            
class Main {
  public static void main(String args[]) {
    // Deklarasi Tipe Data dan Variabel

    int[] nilai = new int[5];

    nilai[0] = 23;
    nilai[1] = 50;
    nilai[2] = 34;
    nilai[3] = 78;
    nilai[4] = 90;

    System.out.println(nilai[0]);
    System.out.println(nilai[1]);
    System.out.println(nilai[2]);
    System.out.println(nilai[3]);
    System.out.println(nilai[4]);
  }
}
Salinlah kode di bawah ini, kemudian jalankan!

Java juga mengizinkan pembuatan array dengan gabungan deklarasi dan inisialisasi nilai awal. Untuk format penulisan ini kita malah tidak butuh proses instansiasi:

Cobalah kode program ini menggunakan compiler
Main.java
                                            
class Main {
  public static void main(String args[]) {
    // Deklarasi Tipe Data dan Variabel

    int[] nilai = {23, 50, 34, 78, 90};

    System.out.println(nilai[0]);
    System.out.println(nilai[1]);
    System.out.println(nilai[2]);
    System.out.println(nilai[3]);
    System.out.println(nilai[4]);
  }
}
Salinlah kode di bawah ini, kemudian jalankan!

Perhatikan perintah di baris 4, disini membuat array nilai dengan tipe data integer (deklarasi) dan langsung memberikan nilai awal (inisialiasi).

Dengan format ini, kita tidak perlu melakukan proses instansiasi, dan Java langsung menetapkan jumlah element array sesuai dengan jumlah nilai yang ingin diinput. Proses pemberian nilai ini dilakukan dengan tanda kurung kurawal {}.

Proses Input Element Array

Setelah array dibuat dan diberi nilai awal, kita tetap bisa mengubah isi dari element array tersebut. Caranya sama seperti mengisi nilai ke dalam variabel biasa, hanya saja sekarang kita harus tentukan nomor index atau key dari array.

Cobalah kode program ini menggunakan compiler
Main.java
                                            
class Main {
  public static void main(String args[]) {
    // Deklarasi Tipe Data dan Variabel

    String[] mahasiswa = {"Muhammad", "Fulan", "Fulanah"};

    mahasiswa[1] = "Ade";

    System.out.println(mahasiswa[0]);
    System.out.println(mahasiswa[1]);
    System.out.println(mahasiswa[2]);
  }
}
Salinlah kode di bawah ini, kemudian jalankan!
