C. Operasi Output Program Java
  • System.out.println()

    • Perintah System.out.println() merupakan perintah untuk menampilkan output kedua yang paling umum digunakan. Perintah System.out.println() dapat menampilkan output dengan ditambah baris baru. Perintah System.out.println() digunakan dengan cara seperti pada kode berikut.

    Cobalah contoh di bawah ini menggunakan compiler
    Main.java
                                    
class Main {
    public static void main(String args[]){

        System.out.println("Semangat Belajar Java");
        System.out.println("di Javatikom");
    }
}
    Dasar-Dasar Pemrograman Java

    Pada materi sebelumnya, kita sudah bisa mencetak "Semangat", "Belajar Java Online", "di Javatikom" ke layar, dan kita menggunakan kode berikut untuk menampilkan "Semangat Belajar Java di Javatikom" .

    Salin kode pada editor ke compiler Trinket, kemudian jalankan!

                                  
class Main {
    public static void main(String args[]){

        System.out.println("Semangat Belajar Java");
        System.out.println("di Javatikom");
    }
}
  • System.out.format()

    • System.out.format() adalah sebuah metode dalam bahasa pemrograman Java yang digunakan untuk menampilkan output yang diformat ke konsol atau output standar. Metode ini mirip dengan metode printf() yang digunakan dalam bahasa pemrograman C. Format output ditentukan oleh string format, yang memungkinkan kita untuk memformat tampilan output seperti menambahkan angka desimal, mengatur panjang string, menambahkan spasi, dan lain sebagainya. Metode ini juga memungkinkan kita untuk memformat dan menampilkan data dengan bahasa dan budaya yang berbeda menggunakan Locale. Perintah System.out.format() digunakan dengan cara seperti kode berikut.

    Cobalah contoh di bawah ini menggunakan compiler
    Main.java
                                    
class Main {
    public static void main(String args[]){

        String namaDepan = "Rizaldy";
        String namaBelakang = "Ervan";

            System.out.format("Nama saya %s %s %n", namaDepan, namaBelakang);
    }
}
    Dasar-Dasar Pemrograman Java

    Pada materi sebelumnya, kita sudah bisa mencetak "Semangat Belajar Java di Javatikom" ke layar, dan kita menggunakan kode berikut untuk menampilkan namaDepan dan namaBelakang .

    Salin kode pada editor ke compiler Trinket, kemudian jalankan!

                                  
class Main {
    public static void main(String args[]){

        String namaDepan = "Rizaldy";
        String namaBelakang = "Ervan";

            System.out.format("Nama saya %s %s %n", namaDepan, namaBelakang);
    }
}

    Perhatikan : di sana kita menggunakan simbol %s untuk mengambil nilai dari variabel di sampingnya. %s artinya string.

    Selain %s, ada juga simbol lain:

    • %d untuk desimal atau angka;
    • %f untuk bilangan pecahan;
    • %n untuk baris baru, bisa juga pakai \n;
