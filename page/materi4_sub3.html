C. Operator Aritmatika

Operator aritmatika adalah operator yang biasa kita temukan untuk operasi matematika. Aritmatika sendiri merupakan cabang ilmu matematika yang membahas perhitungan sederhana seperti kali, bagi, tambah dan kurang.

Selain keempat operasi tersebut, bahasa Java juga memiliki operasi modulo division, atau operator % yang dipakai untuk mencari sisa hasil bagi.

Tabel berikut merangkum operator aritmatika dalam bahasa pemrograman Java:

Operator Keterangan Contoh
+ Operator untuk operasi penjumlahan a = 5 + 2
- Operator untuk operasi pengurangan a = 5 - 2
* Operator untuk operasi perkalian a = 5 * 2
/ Operator untuk operasi pembagian a = 5 / 2
% Operator untuk operasi modulus (perhitungan sisa bagi) a = 5 % 2
++ Operator untuk operasi Increment yaitu menambahkan variabel dengan angka 1 a++ ,++a
-- Operator untuk operasi Decrement yaitu mengurangi suatu variabel dengan angka 1 b-- ,--b

Sebagai contoh buatlah program untuk melakukan operasi artimatika pada nilai variabel x dan variabel y yang bertipe data integer, dengan variabel a diberi nilai 10 dan variabel b diberi nilai 5 seperti pada contoh berikut ini:

Cobalah kode program ini menggunakan compiler
Main.java
                                
class Main {
    public static void main(String args[]){

        // Deklarasi Tipe Data dan Variabel
        int a = 10;
        int b = 5;
        int hasil;

        hasil = a + b;
        System.out.println(hasil);

        hasil = a - b;
        System.out.println(hasil);

        hasil = a * b;
        System.out.println(hasil);

        hasil = a / b;
        System.out.println(hasil);

        hasil = a % b;
        System.out.println(hasil);
    }
}
Salinlah kode di bawah ini, kemudian jalankan!
                                
class Main {
    public static void main(String args[]){

        // Deklarasi Tipe Data dan Variabel
        int a = 10;
        int b = 5;
        int hasil;

        hasil = a + b;
        System.out.println(hasil);

        hasil = a - b;
        System.out.println(hasil);

        hasil = a * b;
        System.out.println(hasil);

        hasil = a / b;
        System.out.println(hasil);

        hasil = a % b;
        System.out.println(hasil);
    }
}

Pada kode di atas terdapat 3 variabel yang bertipe data integer, yang mana variabel a dan b melakukan operasi aritmatika diantaranya a + b, a – b, a * b, a / b, a % b, a++, dan b--. Hasil operasi aritmatika dari variabel a dan b disimpan di dalam variabel hasil. Kemudian ditampilkan menggunakan perintah System.out.println().
